# Глава 10. Больничные листы и другие пособия, оплачиваемые ФСС

В программе реализован полный учет и расчет пособий по временной нетрудоспособности и в связи с материнством, оплачиваемых за счет ФСС (ныне СФР).

## 10.1. Настройка расчета пособий

### Пилотный проект ФСС

При включении **пилотного проекта ФСС** (в учетной политике организации) меняется порядок выплаты пособий:
- Пособия выплачивает напрямую ФСС (СФР)
- Организация только передает в ФСС данные для расчета пособия
- В программе появляются специальные документы и отчеты для взаимодействия с ФСС

### Без пилотного проекта

Организация сама начисляет и выплачивает пособия, а потом возмещает расходы из ФСС.

## 10.2. Документ «Больничный лист»

Основной документ для регистрации нетрудоспособности.

**Возможности документа:**
- Регистрация первичного и продолжения больничного листа
- Автоматический расчет пособия по среднему заработку
- Разделение начисления на часть за счет работодателя (первые 3 дня) и за счет ФСС
- Учет стажа для расчета процента оплаты
- Учет районного коэффициента
- Ограничение максимальным размером пособия
- Возможность ручной корректировки среднего заработка

### Виды нетрудоспособности

- Заболевание или травма (в т.ч. производственная)
- Карантин
- Протезирование
- Санаторно-курортное лечение
- Уход за больным ребенком / взрослым членом семьи
- и др.

## 10.3. Средний заработок для расчета пособий

Программа автоматически собирает данные о заработке за предыдущие 2 календарных года.

При необходимости данные можно:
- Ввести вручную (если сотрудник недавно принят)
- Скорректировать
- Загрузить из справки о сумме заработка (при приеме от предыдущего работодателя)

**Важно**: Для корректного расчета среднего заработка необходимо своевременно вводить все документы начисления зарплаты за предыдущие периоды.

## 10.4. Единовременные пособия за счет ФСС

Документ **Единовременные пособия за счет ФСС** используется для начисления:
- Пособия при рождении ребенка
- Пособия женщинам, вставшим на учет в ранние сроки беременности

Эти пособия **не рассчитываются** автоматически и вводятся отдельным документом.

## 10.5. Пособие по уходу за ребенком до 1,5 лет

Документ **Отпуск по уходу за ребенком**:
- Регистрирует отпуск по уходу за ребенком
- Назначает ежемесячное пособие по уходу за ребенком до 1,5 лет
- Позволяет работать на условиях неполного рабочего времени с сохранением пособия

Пособие рассчитывается автоматически при окончательном расчете зарплаты.

## 10.6. Взаимодействие с ФСС (при пилотном проекте)

### Документы и отчеты

- **Реестр сведений, необходимых для назначения и выплаты пособий**
- **Запрос в ФСС** (для получения информации о застрахованном лице)
- **Ответ на запрос ФСС**

### Порядок работы

1. Сотрудник предоставляет больничный лист (электронный или бумажный)
2. В программе создается документ **Больничный лист**
3. Формируется реестр для передачи в ФСС
4. ФСС рассчитывает и выплачивает пособие напрямую сотруднику

## 10.7. Отчеты по пособиям

- **Пособия за счет ФСС**
- **Реестр пособий**
- **Анализ начисленных пособий**
- **Справка-расчет для возмещения расходов из ФСС** (при непилотном проекте)

---

**Примечание**: При использовании электронных больничных листов рекомендуется подключить сервис **1С:DirectBank** или использовать возможности прямого обмена с ФСС через 1С-Отчетность.
