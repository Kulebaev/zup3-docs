# Глава 23. Особенности базовой версии

Базовая версия программы **1С:Зарплата и управление персоналом 8** имеет ряд ограничений по сравнению с ПРОФ-версией.

## 23.1. Основные ограничения базовой версии

### По количеству организаций

- В базовой версии можно вести учет **только по одной организации**.
- В ПРОФ-версии — неограниченное количество организаций в одной информационной базе.

### По функционалу

В базовой версии **недоступны** следующие возможности:

- **Учет по нескольким организациям** в одной базе
- **Обособленные подразделения**, зарегистрированные в ИФНС как отдельные КПП
- **Учет по территориям**
- **Статьи финансирования**
- **Индексация заработка** (в полном объеме)
- **Использование подработок** (в полном объеме)
- **Работа в модели сервиса** (1С:Фреш)
- **Расширение конфигурации** (ограничено)
- **Некоторые виды обмена** с другими программами 1С

### По администрированию

- Ограничены возможности по настройке прав доступа
- Ограничена работа с дополнительными реквизитами и сведениями
- Нет полноценного разделения ролей «Кадровик» и «Расчетчик»

## 23.2. Когда достаточно базовой версии

Базовая версия подходит для:
- Небольших организаций (до 10–15 сотрудников)
- Индивидуальных предпринимателей
- Организаций без обособленных подразделений
- Случаев, когда не требуется сложный учет (территории, статьи финансирования, индексация и т.д.)

## 23.3. Переход с базовой версии на ПРОФ

При необходимости можно перейти с базовой версии на ПРОФ-версию:
- Перенос данных выполняется с помощью обработки **Перенос данных**
- Переносится почти вся информация (сотрудники, начисления, документы)
- После перехода становятся доступны все возможности ПРОФ-версии

**Рекомендация**: Если планируется рост организации или усложнение учета, лучше сразу начинать работу в ПРОФ-версии.
