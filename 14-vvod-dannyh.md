# Глава 14. Ввод данных для расчета зарплаты

Некоторые начисления зависят не только от относительно постоянных значений (тарифная ставка, процент надбавки), но и от значений, которые меняются из месяца в месяц и становятся известны перед окончательным расчетом зарплаты.

Например:
- Процент разовой/квартальной/годовой премии
- Фактическое выполнение плана
- Выручка от реализации
- Объем сдельных работ

В программе для ввода таких значений используются документы из журнала **Данные для расчета зарплаты**.

## Возможности документов «Данные для расчета зарплаты»

- Вводить значения показателей, от которых зависят начисления
- Вводить данные о рабочем времени
- Использовать начисления, которые будут рассчитаны **только если** на этот месяц введено значение определенного показателя (иначе начисление не попадет в расчет)

Это позволяет не назначать сотруднику в качестве плановых начисления, которые зависят только от ежемесячно вводимых разных значений.

## 14.1. Премии

При начальной настройке можно отметить, что на предприятии начисляется разовая, квартальная или годовая премия, и что ее начисление выполняется при окончательном расчете зарплаты (а не в межрасчетный период).

Тогда становятся доступны следующие виды документов:

- **Ввод процента разовой премии**
- **Ввод процента квартальной премии**
- **Ввод процента годовой премии**
- **Ввод разовой премии** (суммой)
- **Ввод квартальной премии** (суммой)
- **Ввод годовой премии** (суммой)

В документе указывается:
- Месяц, при окончательном расчете которого будет начислена премия
- Подразделение (не обязательно)
- Список сотрудников и размер премии (процент или сумма)

При окончательном расчете за указанный месяц выбранным сотрудникам будет выполнено соответствующее начисление премии.

## 14.2. Доплаты за выполнение плановых показателей

### 14.2.1. Доплата за выполнение плана

**Назначение плана**
Документ **Назначение плана** устанавливает плановое значение для сотрудников выбранного подразделения на неограниченный срок (с указанием периода начала действия).

**Выполнение месячного плана**
Документ **Выполнение месячного плана** регистрирует фактические результаты сотрудников за месяц.

При окончательном расчете происходит соотнесение факта с планом. При перевыполнении плана рассчитывается доплата:
```
(Факт / План) * Расчетная база
```

### 14.2.2. Доплата за выручку от реализации

**Назначение процента доплаты за выручку**
Документ устанавливает процент доплаты для всей организации на неограниченный срок.

**Выручка от реализации**
Документ **Выручка от реализации** регистрирует выручку, приходящуюся на каждого сотрудника за месяц.

При окончательном расчете доплата рассчитывается как:
```
Выручка * ПроцентДоплатыЗаВыручку / 100
```

## 14.3. Сдельные работы

Если сотруднику назначено плановое начисление **Сдельный заработок**, то результат может зависеть от фактически выполненных работ.

**Регистрация выполненных работ**
Документ **Сдельные работы** (из журнала **Данные для расчета зарплаты**).

В документе указываются:
- Сотрудники со сдельной оплатой труда
- Вид выполненной работы (выбирается из справочника)
- Объем выполненных работ
- День выполнения (при необходимости)

При окончательном расчете начисления **Сдельный заработок** сумма определяется как:
```
Объем работ × Расценка (действовавшая на момент работы)
```

Если сумма сдельного заработка окажется меньше оплаты по тарифу за этот период, то будет начислена оплата по тарифу (гарантия).

---

**Примечание**: Функциональность документа **Сдельные работы** может быть изменена пользователем путем настройки соответствующего шаблона ввода исходных данных (см. Приложение 1).
