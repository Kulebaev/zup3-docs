# Глава 22. Прочая отчетность

Помимо основной отчетности по НДФЛ, страховым взносам и персонифицированному учету, в программе формируется ряд других отчетов.

## 22.1. Статистическая отчетность

### Форма П-4 (сведения о численности и заработной плате)

- Формируется в разделе **Отчетность, справки → Статистика**
- Автоматически заполняется на основании данных кадрового учета и расчета зарплаты
- Есть возможность ручной корректировки

### Другие статистические формы

Программа поддерживает формирование:
- Формы 1-Т (условия труда)
- Формы о травматизме
- Других отраслевых статистических форм (в зависимости от вида деятельности)

## 22.2. Отчетность в военкомат

При ведении воинского учета формируются:
- **Список граждан для постановки на воинский учет**
- **Список граждан для снятия с воинского учета**
- **Сведения о гражданах, подлежащих воинскому учету**
- **Карточки воинского учета** (форма Т-2 ВС)

## 22.3. Справки для сотрудников

Из карточки сотрудника или из соответствующих документов можно сформировать:
- **Справка о сумме заработка** (для расчета пособий)
- **Справка 2-НДФЛ** (для сотрудника)
- **Справка о доходах** (произвольная)
- **Расчетный листок**
- **Справка о стаже** (в т.ч. льготном)

## 22.4. 1С-Отчетность

Рабочее место **1С-Отчетность** позволяет:
- Формировать и отправлять регламентированную отчетность в ФНС, ПФР (СФР), ФСС, Росстат и другие органы
- Получать требования и уведомления от контролирующих органов
- Вести переписку с ФНС
- Подписывать документы электронной подписью

## 22.5. Аналитические отчеты по зарплате

Основные аналитические отчеты:
- **Анализ начислений и удержаний**
- **Полный свод начислений, удержаний и выплат**
- **Задолженность по зарплате**
- **Анализ ФОТ**
- **Распределение ФОТ по статьям затрат**
- **Среднесписочная численность**
- **Текучесть кадров**

---

**Рекомендация**: Для большинства организаций основной объем отчетности закрывается через рабочее место **1С-Отчетность**. Статистические и внутренние аналитические отчеты формируются по мере необходимости.
