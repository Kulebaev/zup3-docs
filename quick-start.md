# Быстрый старт для разработчика 1С:ЗУП 3

Этот раздел поможет быстро сориентироваться в типичных задачах при разработке и доработке конфигурации **1С:Зарплата и управление персоналом, редакция 3**.

## 1. Как создать новое начисление

**Основные шаги:**

1. Зайди в **Настройка → Начисления** (или **Зарплата → Начисления**).
2. Создай новый элемент.
3. Выбери **Назначение** начисления (самое важное!).
4. Укажи **Способ выполнения**.
5. Настрой формулу на закладке **Формула расчета**.
6. Настрой закладки:
   - **Расчет базы**
   - **Приоритет**
   - **Зависимости**
   - **Налоги, взносы, бухучет**
   - **Средний заработок**

**Полезные ссылки:**
- Подробно: [Приложение 1. Настройка начислений и удержаний](pr1-nastroika-nachisleniy.md)
- Примеры формул: см. Главу 15

## 2. Как настроить сложную формулу расчета

**Пример:** Доплата за работу в ночное время

```
СтоимостьЧаса * ВремяВЧасахНочное * ПроцентДоплаты / 100
```

**Где взять показатели:**
- `СтоимостьЧаса` — предопределённый показатель
- `ВремяВЧасахНочное` — нужно создать показатель или использовать существующий
- `ПроцентДоплаты` — можно сделать показателем начисления или константой

**Рекомендация:** Используй функцию `?(Условие, ЗначениеЕслиИстина, ЗначениеЕслиЛожь)` для сложной логики.

## 3. Как сделать перерасчет зарплаты за прошлый период

**Варианты:**

| Ситуация                        | Что делать                                      | Документ |
|--------------------------------|--------------------------------------------------|----------|
| Ошибка в одном документе       | Исправить документ                              | Кнопка **Исправить** |
| Нужно пересчитать весь месяц   | Создать документ **Перерасчет**                 | Перерасчет |
| Изменились данные за прошлый месяц | Провести **Начисление зарплаты и взносов** за текущий месяц | Автоматически |

**Подробно:** [Глава 21. Исправление прошлых периодов](21-ispravlenie-periodov.md)

## 4. Как вывести данные по зарплате в свой отчет

**Основные регистры для запросов:**

- `РегистрРасчета.Начисления` — все начисления
- `РегистрРасчета.Удержания` — удержания
- `РегистрНакопления.ВзаиморасчетыССотрудниками` — взаиморасчеты
- `РегистрНакопления.ОтражениеЗарплатыВБухучете` — данные для бухучета

**Пример простого запроса:**
```bsl
ВЫБРАТЬ
    Начисления.Сотрудник,
    Начисления.ФизическоеЛицо,
    Начисления.Начисление,
    Начисления.Результат
ИЗ
    РегистрРасчета.Начисления КАК Начисления
ГДЕ
    Начисления.ПериодДействия МЕЖДУ &НачалоМесяца И &КонецМесяца
```

## 5. Как настроить обмен с банком по зарплатному проекту

1. Создай **Зарплатный проект** (справочник).
2. Укажи формат обмена с банком.
3. В личных данных сотрудников заполни **Номер лицевого счета**.
4. В ведомости **Ведомость в банк** сформируй реестр.
5. Загружай подтверждения через документ **Подтверждение зачисления зарплаты**.

**Подробно:** [Глава 17. Обмен с банками](17-obmen-s-bankami.md)

## 6. Как правильно настроить НДФЛ и взносы

**Важные настройки:**

- В карточке начисления → закладка **Налоги, взносы, бухучет**
- Статус налогоплательщика у физического лица
- Территориальные условия и условия труда (при использовании)
- Учетная политика организации (тариф взносов)

**Подробно:**
- [Глава 18. НДФЛ](18-ndfl.md)
- [Глава 19. Страховые взносы](19-vznosy.md)

## 7. Как добавить новое удержание (например, профсоюзные взносы)

1. Создай вид удержания в **Настройка → Удержания**.
2. Укажи **Назначение** = "Прочие удержания".
3. Настрой формулу (обычно процент от расчетной базы).
4. Назначь удержание сотруднику через документ **Назначение удержания** или в кадровых документах.

## 8. Как работать с индексацией заработка

1. Включи возможность **Индексация заработка** в настройках.
2. Создай документ **Индексация заработка**.
3. Укажи коэффициент и какие показатели индексировать.
4. При необходимости создай **Индексация штатного расписания**.
