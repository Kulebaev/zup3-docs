# Глава 17. Обмен электронными документами с банками в рамках зарплатных проектов

Программа поддерживает электронный обмен с банками в рамках зарплатных проектов. Это позволяет формировать реестры перечислений и получать подтверждения зачисления зарплаты.

## 17.1. Подключение зарплатного проекта

### Настройка в программе

1. В карточке организации или подразделения указывается **зарплатный проект**.
2. В справочнике **Зарплатные проекты** создается запись с указанием банка и формата обмена.
3. В личных данных сотрудников (ссылка **Выплата зарплаты**) заполняется **номер лицевого счета**.

При необходимости можно указать разные зарплатные проекты для разных подразделений.

### Форматы обмена

Программа поддерживает несколько форматов обмена (в зависимости от банка):
- Стандартный формат 1С
- Форматы конкретных банков (Сбербанк, ВТБ и др.)
- Универсальный формат

## 17.2. Формирование реестра перечислений

Из документа **Ведомость в банк** можно сформировать:
- **Реестр перечислений по зарплатному проекту** (для отправки в банк)
- Печатную форму реестра

Реестр формируется в электронном виде в формате, согласованном с банком.

## 17.3. Загрузка подтверждений из банка

После зачисления зарплаты банк может предоставить подтверждение. В программе предусмотрена загрузка таких подтверждений.

### Документ «Подтверждение зачисления зарплаты»

- Загружается файл подтверждения из банка
- Автоматически заполняются суммы зачисленных и незачисленных сумм
- Формируется документ **Сведения о незачисленной зарплате** по сотрудникам, которым зарплата не зачислена

### Сведения о незачисленной зарплате

Используется для:
- Повторной выплаты (в другую ведомость)
- Депонирования зарплаты
- Анализа причин незачисления

## 17.4. Обмен с банком по исполнительным документам

При наличии исполнительных листов программа поддерживает:
- Формирование реестров для перечисления удержанных сумм получателям через банк
- Учет вознаграждения банка-платежного агента

## 17.5. Отчеты

- **Реестры перечислений по зарплатному проекту**
- **Подтверждения зачисления зарплаты**
- **Сведения о незачисленной зарплате**
- **Анализ задолженности по зарплате**

---

**Примечание**: Конкретный порядок обмена (форматы файлов, порядок подписания, способ передачи) необходимо уточнять в банке, с которым заключен зарплатный проект. В программе реализованы наиболее распространенные сценарии. При необходимости форматы можно дорабатывать.
