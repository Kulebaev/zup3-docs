# Приложение 1. Настройка начислений и удержаний

В программе предусмотрена возможность регистрации сумм по широкому спектру видов начислений, а также регистрации выплат доходов для различных целей.

## Список начислений

Большинство видов начислений сосредоточены в списке начислений. Начисления из этого списка предназначены не только для регистрации сумм, но могут использоваться и для регистрации отработанного и не отработанного сотрудниками времени.

Это такие начисления, как:
- Повременная и сдельная оплата труда
- Премии, вознаграждения, стимулирующие выплаты
- Доплаты, надбавки, компенсации
- Натуральные доходы и т. п.

Суммы таких начислений можно не только вводить вручную, но и автоматически рассчитывать по формулам (в том числе сложным), с зависимостью от сумм других начислений и с учетом сведений об отработанном и неотработанном времени.

Для таких начислений можно указать:
- Как они должны учитываться при расчете среднего заработка
- НДФЛ
- Страховые взносы
- При формировании сведений для бухгалтерских программ

В список начислений входят также и **неоплачиваемые отсутствия**:
- Отпуска без сохранения платы
- Прогулы
- Неявки
- Неоплачиваемые простои и т. п.

Они регистрируются в программе только с целью учета времени.

### Начисления, не представленные в общем списке

Часть видов начислений, которые в программе можно произвести сотрудникам, **не представлена** в списке начислений:

- Пособия ФСС при рождении ребенка и при постановке на учет в ранние сроки беременности (регистрируются документом **Единовременные пособия за счет ФСС**)
- Начисления в рамках договоров гражданско-правового характера (договор на выполнение работ, оказание услуг, авторского заказа)

### Незарплатные доходы

Другим видом регистрируемых сумм являются суммы доходов сотрудников, напрямую не связанные с трудовыми отношениями:
- Стоимость призов и подарков
- Выплаты бывшим сотрудникам
- Прочие доходы (аренда и т. п.)
- Дивиденды

Они регистрируются только с целью исчисления НДФЛ и взносов, **не включаются** в ведомость на выплату и в аналитические отчеты по зарплате.

## Назначение начисления

**Назначение** позволяет программе определять:
- Какие настройки являются допустимыми для этого начисления
- Каким образом и где это начисление может назначаться, рассчитываться и выполняться

Список возможных назначений является **закрытым** (расширить его нельзя).

### Специализированные назначения

Некоторые назначения доступны только при выборе соответствующей возможности программы. Они используются только специализированными «межрасчетными» документами:

**Для документа «Больничный лист»**:
- Оплата больничного листа
- Оплата больничного листа за счет работодателя
- Оплата больничного листа, несчастный случай на производстве
- Оплата больничного листа, профзаболевание
- Болезнь без оплаты
- Доплата до среднего заработка за дни болезни
- Отпуск по беременности и родам
- Отпуск по беременности и родам без оплаты

**Для документа «Отпуск»**:
- Оплата отпуска
- Компенсация отпуска
- Материальная помощь к отпуску
- Отпуск (за счет ФСС) на период санаторно-курортного лечения

**Для документа «Простой сотрудников»**:
- Простой по вине работника
- Оплата простоя по вине работодателя
- Оплата простоя по не зависящим от работодателя причинам

**Для документа «Прогул, неявка»**:
- Неявка по невыясненным причинам
- Прогул

**Для других документов**:
- Оплата командировки
- Оплата времени сохраняемого среднего заработка
- Оплата дней ухода за детьми-инвалидами
- Отпуск без оплаты
- Отгул
- Компенсация морального вреда
- Выходное пособие

### Особые назначения

- **Районный коэффициент** и **Северная надбавка** — в информационной базе может присутствовать только по одному начислению с такими назначениями. Назначаются и редактируются в кадровых документах.
- **Надбавка за вредность** — особенным образом назначается и редактируется в кадровых документах.
- **Сдельная оплата труда** — может выполняться только при окончательном расчете зарплаты.
- **Доплата за совмещение** и **Доплата до среднего заработка** — могут назначаться только соответствующими специализированными документами.
- **Пособие по уходу за ребенком до полутора лет** и **Пособие по уходу за ребенком до трех лет** — могут назначаться только соответствующими специализированными документами.
- **Оплата труда в натуральной форме** — может назначаться только в качестве планового начисления.

### Универсальные назначения

Универсальными назначениями, допускающими дополнительное управление, являются:

- **Повременная оплата труда и надбавки** — для описания начислений, составляющих фонд оплаты труда.
- **Компенсационные выплаты** — для описания компенсаций (не учитываются при расчете общего среднего заработка).
- **Материальная помощь** — имеет особенности в обложении НДФЛ, взносами и учете в среднем заработке.
- **Премия** — особым образом учитывается при расчете общего среднего заработка.
- **Доход в натуральной форме** — регистрируются суммы, соответствующие доходу, полученному не в денежной форме.

Для начислений с этими назначениями можно указать **способ их выполнения**.

## Способы выполнения начисления

Для начислений с универсальными назначениями можно указать способ выполнения:

| Способ выполнения | Описание |
|-------------------|----------|
| **Ежемесячно** | Можно назначить плановым. Автоматически выполняется при окончательном расчете зарплаты. |
| **По отдельному документу** | Выполняется только межрасчетными документами (**Разовое начисление**, **Материальная помощь**, **Премия** и т.д.). |
| **В перечисленных месяцах** | Совпадает с «Ежемесячно», но можно выбрать конкретные месяцы. |
| **Только если введено значение показателя** | Выполняется при окончательном расчете только если на этот месяц введено значение определяющего показателя. |
| **Только если введен вид учета времени** | Выполняется только если зарегистрировано время выбранного вида. |
| **Только если время приходится на праздничные дни** | Выполняется только если рабочее время выпадает на государственные праздники. |

## Формула расчета

Для некоторых назначений алгоритм расчета регламентирован законодательством и заложен внутри программы (например, **Оплата больничного листа**). Для них нельзя задать произвольную формулу.

Для других назначений допустимо описать способ расчета **произвольной формулой**.

### Элементы формулы

Формула состоит из показателей, которые соединяются арифметическими операциями (`+`, `-`, `*`, `/`), также могут использоваться скобки и постоянные числа.

Поддерживаются функции:
- `Макс(Выражение1, …, ВыражениеN)`
- `Мин(Показатель1, …, ПоказательN)`
- `Окр(Выражение, Разрядность)`
- `Цел(Выражение)`
- `?(Условие, Выражение1, Выражение2)`

## Показатели расчета зарплаты

Показатели делятся на несколько типов по назначению:

### По объектам ввода значений
- Для каждого сотрудника
- Для всех сотрудников определенного подразделения
- Для всей организации

### По типу
- **Денежный**
- **Числовой**
- **Числовой, зависящий от другого показателя**
- **Числовой, зависящий от стажа**

### По характеру использования
- **Во всех месяцах после ввода значения (постоянное использование)** — например, оклад, процент ежемесячной надбавки.
- **Вводится единовременно документом ввода данных для расчета зарплаты** — например, процент разовой премии.
- **Накапливается по отдельным значениям** — например, количество заключенных договоров.
- **Вводится при расчете, непосредственно в документе начисления**.

## Шаблоны ввода исходных данных

Настройка шаблона позволяет определить вид и функциональность документа **Данные для расчета зарплаты**.

В шаблоне можно настроить:
- Какие показатели будут вводиться.
- Будет ли документ заполняться автоматически.
- Нужно ли вводить значения для нескольких объектов сразу.
- Доступность дат и т.д.
