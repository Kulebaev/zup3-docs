# Глава 11. Учет времени

В программе ведется учет рабочего времени сотрудников. Это позволяет:
- Автоматически рассчитывать повременную оплату труда
- Формировать табель по унифицированной форме Т-13
- Корректно рассчитывать средний заработок

## 11.1. Производственный календарь

В программе хранится **производственный календарь**, который используется для:
- Автоматического заполнения графиков работы
- Определения нормы времени
- Расчета стоимости часа/дня при пересчете тарифных ставок

В поставке уже заполнен календарь по законодательству РФ (пятидневка с учетом праздников и переносов). Можно использовать несколько календарей (например, для разных регионов).

## 11.2. Графики работы

Общие графики описываются в справочнике **Графики работы сотрудников**.

Можно настроить:
- Пятидневка, шестидневка, сменный график, суммированный учет и т.д.
- Неполное рабочее время
- Работу в ночные и вечерние часы
- Дополнительные перерывы для кормления ребенка
- Собственные виды времени (например, "Время на линии", "Время ремонта" и т.п.)

После настройки график можно заполнить автоматически на год. При необходимости — отредактировать вручную.

### Индивидуальные графики

Документ **Индивидуальный график** позволяет временно изменить график работы конкретного сотрудника на месяц (например, поменять местами рабочие и выходные дни при сменной работе).

## 11.3. Работа в выходные и праздничные дни

Возможны две ситуации:

1. **Внеплановая работа** (по приказу) — регистрируется документом **Работа в выходные и праздники**.
2. **Рабочий день по графику попал на праздник** — автоматически учитывается при начислении зарплаты через начисление **Доплата за работу в праздничные дни**.

Оплата может производиться в повышенном размере или с предоставлением отгула.

## 11.4. Работа сверхурочно

Документ **Работа сверхурочно** регистрирует факт сверхурочной работы.

Оплата производится при начислении зарплаты:
- Первые 2 часа — не менее чем в 1,5 размере
- Последующие часы — не менее чем в двойном размере

Можно выбрать компенсацию деньгами или отгулом.

## 11.5. Регистрация переработок (суммированный учет)

При суммированном учете рабочего времени переработки рассчитываются и регистрируются документом **Регистрация переработок**.

Оплата переработок производится в последнем месяце учетного периода с помощью специального начисления.

При увольнении можно оплатить переработки, накопленные после последнего закрытого учетного периода.

## 11.6. Уточнение фактически отработанного времени

### Документ «Табель»

Используется, когда нужно уточнить фактически отработанное время по сравнению с графиком:
- Добавить/убрать часы
- Зарегистрировать ночные, праздничные, сверхурочные часы
- Указать работу на разных территориях или в разных условиях труда в течение одного дня

**Важно**: Для оплачиваемых отсутствий (отпуска, больничные и т.д.) обязательно вводить соответствующие первичные документы. Табель используется для уточнения явки/неявки.

### Документ «Данные для расчета зарплаты»

Можно настроить собственные шаблоны для ввода времени (в том числе сводно за месяц без детализации по дням). Удобно, когда разные сотрудники отвечают за ввод разных видов времени.

## 11.7. Отчет «Табель учета рабочего времени» (Т-13)

Отчет формируется автоматически на основании:
- Назначенного графика работы (общего или индивидуального)
- Зарегистрированных отклонений (отпуска, больничные, сверхурочные и т.д.)
- Данных табеля (если вводился)

---

**Рекомендация**: Для большинства организаций достаточно правильно настроить графики работы и своевременно вводить документы отклонений. Табель используется в основном при сложном учете времени или когда есть выделенные табельщики.
