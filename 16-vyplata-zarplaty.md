# Глава 16. Выплата зарплаты

После расчета зарплаты программа позволяет подготовить ведомости для выплаты начисленных сумм (через банк, кассу или раздатчика).

## 16.1. Места выплаты зарплаты

Зарплата может выплачиваться:
- На карточку в рамках **зарплатного проекта**
- На **произвольный банковский счет** сотрудника
- Через **кассу**
- Через **раздатчика** (обычно руководители подразделений)

Место выплаты можно указать:
- Для всей организации
- Для конкретного подразделения
- Для отдельного сотрудника

Это позволяет автоматически заполнять разные ведомости для разных групп сотрудников.

## 16.2. Ведомости на выплату

В программе предусмотрены отдельные виды ведомостей:

| Вид ведомости                        | Назначение                              |
|--------------------------------------|-----------------------------------------|
| **Ведомость в банк**                 | Зарплатный проект                       |
| **Ведомость перечислений на счета**  | Произвольные счета сотрудников          |
| **Ведомость в кассу**                | Выплата наличными через кассу           |
| **Ведомость выплаты через раздатчика** | Выплата через уполномоченное лицо     |

### Что можно выплатить по ведомости

- **Аванс** (расчет за первую половину месяца, фиксированная сумма или процент от тарифа)
- **Межрасчетные выплаты** (отпускные, больничные, премии и т.д.)
- **Зарплата за месяц** (погашение всей задолженности)

## 16.3. Особенности заполнения ведомостей

При заполнении ведомости учитываются:
- Место выплаты, указанное для сотрудника/подразделения/организации
- Месяц выплаты
- Подразделение (при необходимости)
- Статьи финансирования (при их использовании)
- Ранее исчисленный, но еще не удержанный НДФЛ

Можно настроить **округление сумм** и **процент выплаты** (например, выплатить только 50% от рассчитанной суммы).

## 16.4. Выплата через банк (зарплатный проект)

- Указывается зарплатный проект
- Должны быть заполнены номера лицевых счетов сотрудников
- При использовании электронного обмена формируется файл для банка
- Поддерживается загрузка подтверждений зачисления из банка

Если банк не поддерживает подтверждения, можно использовать документ **Сведения о незачисленной зарплате**.

## 16.5. Выплата через кассу и раздатчика

- Формируются печатные формы Т-53 или Т-49
- Ведомость считается оплаченной при проведении (без дополнительных документов)
- При неявке сотрудника можно оформить **Депонирование зарплаты**

## 16.6. Учет депонентов

Программа позволяет вести учет депонированной зарплаты:
- **Депонирование зарплаты**
- **Выплата депонированной зарплаты**
- **Списание депонированной зарплаты** (по истечении срока исковой давности)

Можно формировать **Книгу учета депонентов** и аналитические отчеты.

## 16.7. Автоматическое формирование ведомостей

Из многих документов (отпуск, больничный, премия и т.д.) есть команда **Выплатить**. Открывается окно, где можно сразу создать необходимые ведомости с учетом мест выплаты сотрудников.

## 16.8. Компенсация за задержку зарплаты

Документ **Компенсация за задержку выплаты заработной платы** рассчитывает компенсацию исходя из ставки рефинансирования ЦБ (или другой ставки, заданной в регистре).

## 16.9. Анализ задолженности

Отчет **Задолженность по зарплате** показывает взаиморасчеты с сотрудниками (с точки зрения расчетчика зарплаты).

При необходимости вернуть излишне выплаченные суммы можно использовать документ **Возврат сотрудником задолженности**.

---

**Важно**: Выплаты бывшим сотрудникам и прочим лицам регистрируются специальными документами только для целей НДФЛ и взносов. Взаиморасчеты с ними в программе не ведутся.
