# Глава 15. Расчет зарплаты и взносов

Выполнение окончательного расчета зарплаты в программе предполагается по итогам каждого месяца в целом за весь этот месяц.

Для этого предусмотрен документ **Начисление зарплаты и взносов**. Он заполняется и рассчитывается полностью автоматически согласно всем зарегистрированным в программе сведениям.

## Структура документа «Начисление зарплаты и взносов»

Документ содержит несколько закладок:

- **Начисления** — начисления штатным сотрудникам за текущий месяц + доначисления/перерасчеты за предыдущие месяцы
- **Договоры** — вознаграждения по договорам ГПХ
- **Пособия** — пособия по уходу за детьми
- **Удержания** — исполнительные листы, профсоюзные взносы, добровольные страховые взносы и т.д.
- **НДФЛ** + **Корректировки выплаты**
- **Займы** — погашение займов и материальная выгода
- **Взносы** — страховые взносы

## 15.1. Начисления и доначисления, перерасчеты

Закладка **Начисления** заполняется автоматически с учетом:
- Плановых начислений, назначенных сотрудникам
- Начислений, по которым в этом месяце зарегистрировано время
- Начислений, по которым в этом месяце введены значения показателей

Начисления заполняются:
- Отдельно по каждому рабочему месту сотрудника
- Отдельно по основному месту и совместительствам
- Отдельно по подработкам (при их наличии)

### Приоритет начислений

При заполнении и расчете учитывается **приоритет** начислений. Начисления с более высоким приоритетом вытесняют начисления с более низким приоритетом.

Даже если результат начисления с регистрацией отработанного времени получается нулевым, его наличие в документе может влиять на последующий расчет среднего заработка.

## 15.1.1. Повременная и сдельная оплата

### Оплата по окладу
**Формула:**
```
Оклад * ДоляНеполногоРабочегоВремени * ВремяВДнях / НормаДней
```

### Оплата по окладу (по часам)
**Формула:**
```
Оклад * ВремяВЧасах / НормаЧасов
```

### Оплата по часовому / дневному тарифу
Простое умножение ставки на отработанное время.

### Сдельный заработок
```
Макс(СдельныйЗаработок, Оклад * ... )   // или по часовому тарифу
```

Гарантируется минимальная оплата по тарифу, если сдельный заработок оказался меньше.

### Командировка
Рассчитывается на основе среднего заработка, зарегистрированного в документе **Командировка**.

## 15.1.2. Надбавки, доплаты и премии

### Районный коэффициент
```
(Макс(РайонныйКоэффициент, 1) – 1) * РасчетнаяБаза
```

### Северная надбавка / Надбавка за вредность / Надбавка за выслугу лет
Процент × Расчетная база (с учетом своих списков базовых начислений).

### Доплата до среднего заработка
Сравнивается средний заработок за период с фактическим заработком сотрудника. Доплачивается разница (если средний больше).

### Доплата за совмещение
Рассчитывается от размера доплаты, указанного в документе **Совмещение должностей**.

### Доплата за выполнение плана / за выручку
Используют данные, введенные документами **Данные для расчета зарплаты** (Назначение плана, Выполнение месячного плана, Выручка от реализации).

### Премии (ежемесячные, разовые, квартальные, годовые)
Могут рассчитываться:
- Процентом от расчетной базы
- Фиксированной суммой

Для квартальных и годовых премий база берется за соответствующий период (предыдущий квартал / год).

## 15.1.3. Работа в ночное время, сверхурочно, в праздники и выходные

Для расчета этих начислений используются показатели **Стоимость часа / дня**, которые зависят от:
- Выбранного порядка пересчета тарифной ставки (по среднемесячной норме, по графику сотрудника или по производственному календарю)
- Совокупной тарифной ставки сотрудника (оклад + надбавки и т.д.)

### Доплата за работу в ночное / вечернее время
```
СтоимостьЧаса * ВремяВЧасах * ПроцентДоплаты / 100
```
(По умолчанию 20% за ночные, 10% за вечерние)

### Оплата сверхурочных
```
(ОтработаноСверхурочноВПределах2Часов * 1.5 + (ОтработаноСверхурочно – ОтработаноСверхурочноВПределах2Часов) * 2) * СтоимостьЧаса
```

### Оплата работы в праздничные и выходные дни
```
ВремяВДняхЧасах * СтоимостьДняЧаса * 2
```
(или без коэффициента 2, если предоставляются отгулы)

## 15.2–15.7. Другие закладки документа

- **Договоры** — автоматически заполняются суммы по договорам ГПХ и актам.
- **Пособия** — пособия по уходу за ребенком до 1,5 и до 3 лет.
- **Удержания** — исполнительные листы, профсоюзные взносы, добровольные страховые взносы и т.д.
- **НДФЛ** — окончательное исчисление налога нарастающим итогом + корректировки выплаты.
- **Займы** — погашение основного долга, процентов и материальная выгода.
- **Взносы** — исчисление страховых взносов нарастающим итогом.

## Важные замечания

- Документ **Начисление зарплаты и взносов** следует вводить **после** всех первичных документов за месяц.
- Можно вводить отдельные документы по подразделениям.
- Результаты расчета можно корректировать вручную (они выделяются жирным шрифтом).
- При частом ручном редактировании рекомендуется отключить автоматический пересчет документов в настройках расчета зарплаты.
- При использовании статей финансирования, территорий и условий труда результаты расчета можно смотреть в соответствующих разрезах.
